# Plain-Language Glossary

A cheat sheet for the words you'll hear. No prior knowledge assumed.
Add your own notes anytime — this is your file.

| Term            | What it actually means                                                                 |
|-----------------|----------------------------------------------------------------------------------------|
| **Folder / Directory** | A container for files. Same thing, two names.                                   |
| **Terminal**    | A way to control the computer by typing commands. Your assistant handles this for you. |
| **Python**      | A beginner-friendly programming language. The language we build your tools in.          |
| **Script**      | A short text file full of instructions the computer runs top to bottom. A mini-program. |
| **AI Agent**    | A small program that uses an AI brain to do a task on its own, start to finish.          |
| **API**         | A way for two programs to talk. It's how your tool "phones" the AI to get an answer.     |
| **API Key**     | A secret password that lets your tool use the AI. Guard it like a credit-card number.    |
| **.env file**   | A private file where secret keys are kept, hidden from sharing or backups.               |
| **Git**         | A "save history" system that tracks every version of your work, so nothing is ever lost. |
| **Repository (repo)** | A project folder that Git is watching and keeping history for.                    |
| **Library / Package** | Pre-built code others wrote that you borrow, so you don't reinvent the wheel.     |
