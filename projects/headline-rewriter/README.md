# Headline Rewriter ✍️

Your first AI tool. You give it one marketing headline; it gives you 5 rewrites,
each in a different style.

## What's in this folder

| File               | What it is                                                        |
|--------------------|-------------------------------------------------------------------|
| `main.py`          | The tool itself. Open it — the comments explain every step.        |
| `requirements.txt` | The list of pre-built "kits" the tool needs.                       |
| `README.md`        | This file.                                                         |

## How to run it

You don't need to memorize this. Just tell your assistant **"run the headline tool"**
and it handles the steps. For reference, here's what happens under the hood:

```
python3 main.py
```

It will ask you to type a headline, then print 5 rewrites.

## What you'll need first

This tool talks to the Claude AI, so it needs your **API key** (the secret password).
If you haven't set that up yet, just ask your assistant — it's a 5-minute, one-time step.
