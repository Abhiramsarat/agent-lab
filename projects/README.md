# Projects

Every tool you build lives in its own folder right here.

This keeps experiments separate — like having a clean workbench for each project
instead of one giant messy desk. If one tool breaks, the others are untouched.

## Starting a new tool

You don't create these by hand. Just tell your assistant something like:

> "Let's start a new project that turns a blog post into tweets."

...and a new folder will appear here, set up and ready to go.

(This folder looks empty right now because you haven't built anything yet —
that's exactly how it should be on day one.)
